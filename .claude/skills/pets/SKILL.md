# Pets — Aloe Property Management (Resident Coordinator)

Every pet request starts with the **Aptly Pet Request form** — never handle one
off-form. Pets require **owner/landlord approval**, carry a **$500
non-refundable** acceptance fee, and are recorded on the **Rentvine lease page**.

> **All pet requests must go through the form:**
> https://portal.getaptly.com/form/zhW2y7KzKGLzdfyqj/new/zaRqCdNo66X67BxST
> — it creates a card on the **Aptly Pet Request board.**

## Adding an approved pet

1. **Tenant submits the Pet Request form** (link above) → card on the Aptly Pet
   Request board.
2. **Check existing pets** — whether they already have a pet, and how many.
3. **Confirm the landlord/owner allows pets** — required **before** approval.
4. **Add the fee:** one-time **$500**, memo **"Pet Acceptance Fee"**, marked
   **MGMT PET FEE NOT REFUNDABLE**. Place the charge for the **1st of the next
   upcoming month.**
5. **Record it:** make a note on the **Rentvine lease page** and **add the pet to
   the animals section.**

## Removing a pet

- Make a note on the **Rentvine lease page** and **remove the pet from the
  animals section.**
- The **pet fee is NOT refundable** — no refund on removal.

## Unauthorized pet (found via HOA, inspection, or violation)

1. Send the tenant the **"Unauthorized Pet on Premises – Action Required"** email.
2. Add a **$75 cure notice fee.**
3. Have them **fill out the Pet Request form** for acceptance.
4. **If approved:** add the **$500 non-refundable pet acceptance fee** and
   complete a **pet addendum.**

## Owner approval gate

- **A pet is never added to the lease without owner approval.**
- **Confirming the landlord allows pets** is required before any approval.

## Service / ESA animals (not pets)

Service animals and emotional-support animals are **not pets** — **no pet fee**,
and they follow the fair-housing / ESA documentation process (see
`lead-intake-response`). Never run a service/ESA animal through the pet-fee path.

## Documentation

Note everything on the **Rentvine lease page**, keep the **Aptly Pet Request
card** updated, and (for unauthorized pets) keep the cure email and addendum on
file.

## What good output looks like

- Request came **through the form / Aptly Pet Request board.**
- **Existing pets checked**, **landlord-allows-pets confirmed**, and **owner
  approval** obtained before adding to the lease.
- **$500 non-refundable Pet Acceptance Fee** charged (memo "Pet Acceptance Fee",
  MGMT PET FEE NOT REFUNDABLE), dated the **1st of next month.**
- Pet **added to the Rentvine animals section** and noted on the lease page.
- Unauthorized pet handled: cure email + **$75 cure fee** + form → if approved,
  **$500 fee + pet addendum.**

## This skill NEVER

- **Never approves a pet without confirming the landlord allows pets.**
- **Never refunds the pet fee.**
- **Never skips the Aptly Pet Request board** — all pet requests go through the
  form.
- **Never adds a pet to the lease without owner approval.**
