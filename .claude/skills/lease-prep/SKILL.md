# Lease Prep & Move-In — Aloe Property Management (Residential)

Scope: from **approved applicant → signed lease + cleared funds → hand-off to property management.** Screening is done; this skill picks up at approval.

## Step 1 — Create the lease in Aptly

1. Click **New/Match Lease → Create New Lease.**
2. **Title:** property address.
3. **Residents:** remove Emergency Contact — show applicant names only.
4. Verify all info matches the application.
5. **Summary Page — confirm all charges:**
   - Rent amount
   - Resident Benefit Package ($49.95/mo)
   - Security Deposit (1x rent)
   - Pet Fee ($250/pet, if applicable)
   - Cleaning Fee ($500 non-refundable)
   - Admin Fee ($250)
6. Click **Confirm and Continue** — this auto-transfers tenants to Rentvine.

## Step 2 — Set up Rentvine

Log into aloepm.rentvine.com after Aptly pushes the tenant over.

**Add Recurring Charges:**
- Rent Income (set posting date = first desired date)
- Resident Benefit Package ($35/mo)

**Add One-Time Charges via Actions → Add Charge (all posted on move-in date):**
- Administrative Fee: $250
- MGMT NR Cleaning Fee: $500
- Security Deposit: 1x rent (varies)
- MGMT Pet Fee NR: $250/pet (if applicable)
- Prorated Rent: required if move-in is not the 1st
  - Formula: Rent ÷ 30 = daily rate × days remaining
  - If moving in on/after the 26th: prorated days + full next month
- Resident Benefit Package: $15 if move-in is 15th or later

**Lease page settings:**
- Verify Move-In Date, Start Date, End Date
- Update NSF amount to **$75.00**

**Send Portal Invitations:**
- Click 3 dots next to each tenant name → Send Portal Invitation

**Allow Earnest Deposit Payment:**
- Lease Info → Actions → Enable "Override System Allowed Payment Amount" → Any Amount
- After tenant pays → return and **disable** the override

## Step 3 — Earnest deposit

- Due **within 24 hours of approval** — non-refundable.
- $1,500 required to secure the property (holds up to 14 days; up to 30 days in slower market).
- Credited toward security deposit once lease is fully signed.
- If not received in 24 hours → **automatic denial**, proceed to next applicant.
- Any amount paid over $1,500 is refundable if they choose not to proceed; the $1,500 is not.

## Step 4 — Lease signing

- Lease sent electronically via Aptly — one email per adult on the lease.
- **Every adult 18+ must sign.**
- Must be signed within **24–48 hours** of being sent.
- Keys/access codes released at **9am for a 10am lease start**, after signing AND cleared funds.
- **Funds before keys — no exceptions.**

## Step 5 — Key handoff & move-in

- Tenant sets up utilities before move-in date (their responsibility).
- Citizen Home Solutions concierge may contact them (they consented in the application).
- Move-in ready inspection completed before move-in date — any outstanding items addressed then.
- **Do NOT promise any repairs or upgrades before move-in** — leasing manager handles with maintenance.

## Cancel move-in (earnest deposit not paid / tenant backs out)

In Rentvine (tenant's page):
1. Remove recurring charges: Rent, RBP, SNRBP
2. Remove all transaction charges:
   - Rent (full or prorated per move-in date)
   - Admin Fee ($250)
   - MGMT NR Cleaning Fee ($500)
   - MGMT Pet Fee NR ($250/pet if applicable)
   - RBP ($35 or $15 if after 15th)
   - SN-RBP (full amount regardless of move-in date)
   - Security Deposit
3. Actions → Change Status → **Closed / Closed**
4. Email owner: original tenant did not pay earnest deposit and did not move forward. Home was NOT taken off market (we don't take it off until earnest is paid).
5. In Aptly → change stage to **Abandoned**

## Key rules

- **Funds before keys** — signed lease + cleared earnest deposit + first month's rent before any key or code is released.
- **Never promise repairs** before move-in — leasing manager + maintenance handle it.
- **Never take the home off market** until earnest deposit is paid and cleared.
- **Never skip the portal invitation** — tenants need portal access to pay rent.
- If tenant wants to add/remove a roommate after signing → contact us first; additional screening + lease addendum required.

## Common lease signing FAQs

- **Signed lease but don't want to move in?** Subject to the lease and lease break fee.
- **Paid more than $1,500 earnest?** Only $1,500 reflected in lease; overage refunded if they don't proceed.
- **Can I review before signing?** Yes — encourage it. Lease copy: https://www.notion.so/Copy-of-Lease-Agreement-25e76555273a808083e4c8aace5a96ff
- **How do I sign?** Electronically via Aptly link sent to each adult's email.
- **When is first rent due?** On or before move-in date; then 1st of every month.
- **Where do I pick up keys?** Instructions sent after signing + cleared funds. Access at 9am for 10am start.
- **Can I do a shorter lease?** Depends on the home and market — ask, it's not guaranteed.
- **Can I do a longer lease?** Must apply and be approved first; owner decides; not recommended.
